# RobUIHeal

RobUIHeal is a World of Warcraft addon focused on performance, original systems, and a self-contained architecture.

The core goal of this project is to build a modern healing UI framework without relying on external addon frameworks or libraries.

All systems are designed specifically for this project with performance, maintainability, and control as the primary goals.

---

## Core Philosophy

RobUIHeal is designed to remain lightweight, efficient, and independent.

The project intentionally avoids external frameworks and libraries in order to maintain full control over architecture and performance.

All systems should be implemented directly inside the project unless explicitly approved otherwise.

---

## External Code Policy

External addons, frameworks, or libraries are **not permitted** unless explicitly approved by the project owner.

This includes but is not limited to:

* Ace3
* AceAddon
* AceGUI
* AceConfig
* Any Ace library
* Any external addon framework
* Any runtime dependency from other addons

The core concept of the addon is that **all code should be original and not reliant on other addons** unless approval has been granted.

Approval of external code is the **sole right and privilege of the project owner**.
